# Cat Shelter Management System - MVP (Version 1)

## Goal
Deliver basic navigation and core cat profile access for day-to-day shelter use.

## 1. Main Page
The application opens on a main page that shows:
- Title: **Friends of Larnaca Cats**
- A list of locations (initially, for example: Foster Evgenia)

User action:
- Click a location to open its location page.

## 2. Location Page
The location page shows:
- List of cats in the selected location
- **Add Cat** button

User actions:
- Open a cat profile
- Add a new cat

## 3. Cat Page
The cat profile page includes these sections:
- Basic information
- Medical section
- Notes
- Attachments
- History

## MVP Scope Summary
Included in Version 1:
- Page-to-page navigation: Main Page -> Location Page -> Cat Page
- View cats by location
- Open a cat profile
- Add a new cat
- Display key profile sections (basic info, medical, notes, attachments, history)
